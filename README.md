# Social Media Sentiment Analysis

## 📌 Project Description
This project analyzes Twitter data to classify sentiment (positive/negative).

## ⚙️ Technologies Used
- Python
- Machine Learning
- Scikit-learn
- NLP (NLTK)

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the project:
   python app.py

## 📊 Output
- Generates result.csv with predictions
